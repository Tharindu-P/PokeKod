# Pokédex

## Installation
```zsh
sudo apt install composer
sudo apt install php-mongodb
sudo composer install
```

##### Suppression du fichier composer.lock
```zsh
sudo rm composer.lock
```

## Usage

##### macOS/Linux:
```zsh
sudo php -S localhost:7777
```
